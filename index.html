How to use


This map viewer includes data on outdoor recreation businesses and non-profits, as well as public lands and recreational facilities. To interact with business and non-profit data, use the tabs and filters on the top of the screen.

Using the Point Clustering Tool

In the top left corner, clicking the button with the symbol, will show summary statistics for the number of businesses or organizations in different areas of the state. Point Clustering responds to any filter selections.

You can organize this data by CEDR Region, Tourism Region, County, or Municipality by clicking on the respective geography on the top of the page. Click the same geography again to turn off the layer and display generic clustering.

The icon resets the view to New Hampshire. The icon re-opens this info panel. The icon opens a print prompt that exports the current extent.
